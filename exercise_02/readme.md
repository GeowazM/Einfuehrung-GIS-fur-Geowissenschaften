# Übung 2
## Ziel der Übung
* Eigene Vektorsignaturen gestalten.
* Eine Karte gestalten und typische Elemente wie Legenden, Nordpfeile und Maßstabsleisten hinzufügen.
* Eine Karte als PDF-Dokument exportieren.
